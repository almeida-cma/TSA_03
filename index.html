Se o hardware é o corpo, o software é a mente. Descubra o mundo dos programas que dão vida aos computadores.

💡 O que é Software?

🧠

Definição

Software é a parte lógica do computador - o conjunto de instruções que diz ao hardware o que fazer.

🔄

Ciclo Básico

Input → Processamento → Output
Entrada de dados, processamento e saída de resultados.

💾

Armazenamento

Permanente: HD/SSD
Durante execução: RAM (volátil)

📊 Classificação dos Softwares

📜 Classificação por Licença

Proprietário

Código Fechado

Criado e distribuído por empresas. Usuários pagam por licenças limitadas.

Exemplos: Microsoft Office, Adobe Photoshop, Windows

Livre / Open Source

Código Aberto

Desenvolvido colaborativamente. Código-fonte acessível e modificável.

Exemplos: Linux, Firefox, LibreOffice, Android

Software livre NÃO é sinônimo de gratuito! A liberdade é sobre o código, não necessariamente o preço.

🎯 Classificação por Finalidade

Sistemas Operacionais

Gerenciam hardware e fornecem interface

Windows, Linux, macOS, Android

Produtividade

Editores de texto, planilhas, apresentações

Word, Excel, PowerPoint, Google Docs

Design Gráfico

Criação e edição de imagens

Photoshop, Illustrator, GIMP, CorelDRAW

Edição de Vídeo

Manipulação de vídeos

Premiere Pro, Final Cut, DaVinci Resolve

Banco de Dados

Armazenamento e gerenciamento de dados

MySQL, Oracle, SQL Server, PostgreSQL

Navegação Web

Acesso à internet

Chrome, Firefox, Edge, Safari

Segurança

Proteção contra ameaças

Avast, Norton, McAfee, Kaspersky

📱 Classificação por Plataforma

Desktop

Computadores pessoais e estações de trabalho

Windows, Photoshop, Word

Web

Executados em servidores, acessados via navegador

Gmail, Google Docs, Trello

Mobile

Smartphones e tablets

Apps Android/iOS, WhatsApp, Instagram

Embarcados

Integrados a dispositivos específicos

Sistemas de carros, eletrodomésticos, equipamentos médicos

⚙️ Classificação por Modo de Execução

Nativo

Compilado para arquitetura específica

C/C++ compilados

Interpretado

Executado por interpretador, sem compilação prévia

Python, JavaScript, PHP

Virtual

Executado em máquina virtual

Java (JVM), C# (.NET)

2

Distribuir

3

Melhorar

Modificar e publicar as melhorias

Etapa Descrição
1. Análise de Requisitos Entender o que o software precisa fazer; coletar dados com usuários
2. Projeto Definir arquitetura, design de interface (UI/UX), modelagem
3. Codificação Escrever o código-fonte na linguagem escolhida
4. Testes Unitários, integração, sistema, aceitação, beta
5. Implantação Empacotar, distribuir, instalar, configurar
6. Manutenção Correção de bugs, patches de segurança, novas funcionalidades
7. Aposentadoria Migrar dados, encerrar suporte, remover software

📜 Evolução das Linguagens de Programação

1950 - Fortran

Primeira linguagem de alto nível; científica/engenharia

1960 - COBOL, PL/I

Aplicações comerciais e governamentais

1970 - C

Criação de sistemas operacionais (Unix)

1980 - C++, Objective-C

Orientação a objetos

1990 - Java, Python

Portabilidade (Java), simplicidade (Python)

2000+ - C#, JS, Swift, Go

Web, mobile, cloud, ciência de dados
